# Insportz Dubai

A modern React + TypeScript website for Insportz Dubai sports facility.

## Features

- **Responsive Design**: Desktop and mobile optimized
- **Modern Navigation**: Animated hamburger menu with dropdown support
- **Sports Facilities**: Indoor Cricket, Padel, Badminton, Volleyball, Football, Table Tennis
- **Smooth Animations**: Framer Motion powered transitions
- **Custom Styling**: SCSS modules with glassmorphism effects

## Tech Stack

- **React 19** with TypeScript
- **Vite** for fast development
- **React Router** for navigation
- **Framer Motion** for animations
- **SCSS Modules** for styling
- **Custom Font**: Sequel100Black

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/         # Page components
├── views/         # View components
├── assets/        # Images, icons, fonts
├── styles/        # Global styles
└── App.tsx        # Main app component
```

## Path Aliases

- `assets/*` → `src/assets/*`
- `components/*` → `src/components/*`
- `pages/*` → `src/pages/*`
- `views/*` → `src/views/*`
- `styles/*` → `src/styles/*`
